Class QueryResultAbstract

Abstract query result wrapper, derived from SplQueue. The constructor should take whatever the database query returns.

Since result record sets usually are used exactly one time, this SplQue objects are set to \SplDoublyLinkedList::IT_MODE_DELETE i.e. after using one element it will be deleted.

To retrieve a 'persistent' result set, getRows() returns a clone that does not delete elements after access.
